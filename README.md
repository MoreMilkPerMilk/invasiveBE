# Invasive backend

### Install requirements with
`python -m pip install -r requirements.txt`

### Run with
`./start.sh`
or
`uvicorn app:app --reload`

### View auto-gen docs @
`http://127.0.0.1:8000/docs`

### Endpoints
- Communities
- Tasks
- Councils
- Locations
- Persons
- Species
- Weeds
- see MOdels/Models.md for more
